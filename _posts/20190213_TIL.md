##### 1. 핀테크 아카데미

- node.js 설치
- JavaScript 기본 문법
- 동기/비동기 프로그래밍 설명
  - 동기: 작업 하나가 끝나면 다음 작업을 진행. 작업 순서가 중요할 때 사용.
  - 비동기: 동시에 작업들을 지시하고 끝날때까지 대기.
- 콜백 함수
- 콜백 지옥을 방지하기 위한 방법 중 하나인 nsync
- npm, express 서버 설치
- 서버와 mysql 연결
- ejs 사용
